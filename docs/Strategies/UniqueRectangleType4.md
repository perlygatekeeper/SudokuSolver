# Unique Rectangle Type 4

Unique Rectangle Type 4 uses a strong link between the two roof cells to
prevent a deadly rectangle.

Four unsolved cells occupy two rows, two columns, and exactly two boxes. Two
floor cells contain only the same pair `{a,b}`. Both roof cells contain `a` and
`b` plus one or more extra candidates.

If one rectangle candidate appears nowhere else in the roof cells' shared row
or column, the roof cells form a strong link for that candidate. The other
rectangle candidate can then be removed from both roof cells.

This deduction assumes the puzzle has a unique solution.
